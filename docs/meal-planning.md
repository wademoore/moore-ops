## Phase 5 Facilitation Steps

### Step 1 — Load inputs
1. Read the Meal Planning Preferences doc (Drive ID above)
2. Read the Recipe Library doc (Drive ID above)
3. Pull the Menu calendar for the coming Monday–Sunday — show which days already have dinner set and which are empty
4. Pull all Moore family calendars for the week to identify busy nights

### Step 2 — Classify each night

| Night type | Meal constraints |
|---|---|
| BUSY (activity, practice, swim meet, etc.) | ≤30 min cook time |
| OPEN weeknight | Any weeknight recipe. House Manager or Robyn cooking. |
| Weekend — Wade cooking | Either/Weekend tagged recipes. More ambitious if calendar is light. |
| Weekend — eating out | Mark as "Eat Out" — no meal or groceries needed (1–2 nights typical) |
| Home Chef night | Adults-only. Kids need a simple parallel option. |

Waves practice runs most evenings during swim season (June–July) — treat any practice night as BUSY.

### Step 3 — Build the plan

Rules:
- Serves 4 (Wade, Robyn, Myles, Ophelia)
- No planned leftovers — fresh meal each night
- Recipe library first — prefer known recipes over new suggestions. Flag any new suggestion clearly.
- Protein variety — avoid same protein two nights in a row
- Pork tenderloin is a family-wide favorite — include roughly every 2–3 weeks
- Spice — keep family meals mild. Wade adds heat to his own.
- Kids' fallback — when the planned meal is adult-leaning (fajitas, crab cakes, Home Chef, ribs), note a simple parallel kid option using the same cooking method where possible
- Cuisine variety — low. Stick to the family's familiar rotation.
- Repeat cadence — same meals can recur every 2–3 weeks. No need to force novelty.
- Kid introduction goal — once every 2–3 weeks, flag one meal as a gentle stretch: "🌱 Kid stretch — may need fallback"
- Deep Indian Kitchen Butter Chicken (frozen) is a valid adults-only pantry night — kids get a quick parallel (quesadillas, hot dogs, SpaghettiOs)

### Step 4 — Present the plan

Output format:

```
MOORE FAMILY DINNER PLAN — Week of [Date]
MON  [Meal name]  [⚡ if ≤30 min]  [HC if Home Chef]
TUE  [Meal name]
WED  [Meal name]
THU  [Meal name]
FRI  [Meal name]
SAT  [Meal name]  — or: Eat Out
SUN  [Meal name]  — or: Eat Out

Notes:
* [Kid fallback notes]
* [Cook assignments — e.g., "Wade grilling Saturday"]
* [Any new recipe flags]
```

Then ask: **"Does this plan work, or any swaps before I add it to the calendar?"**
Wait for confirmation before creating events.

### Step 5 — Create calendar events

After confirmation, create a Google Calendar event for each dinner night (skip Eat Out nights):
- Calendar: Menu calendar (ID above)
- Time: 6:00–7:00 PM ET
- Title: meal name (with HC: prefix if Home Chef)
- Description: cook assignment + kids' alternate if applicable + recipe URL if in library
- Location: restaurant name + address for dining-out nights if known

### Step 6 — Grocery list (if requested)

Only produce if Wade or Robyn asks for it after confirming the plan.

Format:

```
GROCERY LIST — Week of [Date]
🥩 MEAT & SEAFOOD  •  ...
🥛 DAIRY & EGGS  •  ...
🥦 PRODUCE  •  ...
🥫 CANNED & DRY GOODS  •  ...
❄️ FROZEN  •  ...
🍞 BREAD & BAKERY  •  ...
🧴 OTHER / HOUSEHOLD  •  ...
🏪 COSTCO (bulk — consider ordering here)  •  ...
```

Exclusions — never list these (always on hand):
- Oils: EVOO, canola oil, avocado oil spray, balsamic vinegar, white vinegar
- Dairy: whole milk, butter, heavy cream, eggs
- Freezer proteins: bacon, breakfast sausage patties
- Canned: Prego pasta sauce, Del Monte corn, Del Monte carrots, tomato sauce, bone broth
- Spices: salt, pepper, garlic powder, paprika, Lowry's, oregano, parsley, vanilla extract
- Grains: jasmine rice, thin spaghetti, quick oats
- Condiments: ketchup, mustard, BBQ sauce, mayo, Miracle Whip, soy sauce, Heinz Chili Sauce, preserves, Pearl Milling syrup
- Baking: Bisquick, Krusteaz Waffle Mix, bread flour, brown sugar, chocolate chips
- Do NOT include Home Chef meal ingredients — those ship with the kit
- Do NOT include kids' snacks — ordered separately
- Note Costco-appropriate bulk items with 🏪 tag

Primary store: Walmart (delivery). Harris Teeter for specialty items.
