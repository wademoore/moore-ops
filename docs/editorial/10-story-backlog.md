## Backlog Field Definitions

| Field | Description |
|-------|-------------|
| **Working title** | Internal label only — not the published headline |
| **Category** | From [06-story-discovery-playbook.md](06-story-discovery-playbook.md) |
| **Editorial finding** | One sentence: what the data shows |
| **Supporting datasets** | Specific files that support the finding |
| **Status** | `Candidate` / `In progress` / `Published` / `Spiked` |
| **Priority** | `High` / `Medium` / `Low` |
| **Potential graphics** | Suggested visual(s) from [08-art-direction.md](08-art-direction.md) |
| **Publication target** | Which edition type this fits |
| **Notes** | Caveats, open questions, or follow-up needed |
